# 🏏 IPL Cricket Analysis Dashboard

An interactive multi-page Power BI dashboard analyzing IPL cricket 
data to uncover team and player performance insights.

## 📊 Dashboard Pages

### 1. IPL Season Overview
- Total Runs: 330K
- Total Wickets: 13K
- Total Matches: 1,095
- Total Sixes: 13K
- Wickets breakdown by bowling team (donut chart)
- Runs comparison by batting team

### 2. Batting Leaderboard
- Top run scorer: V Kohli
- Top six hitter: CH Gayle
- Total Runs: 329,798 | Total Fours: 29,827 | Total Sixes: 13,036
- Batting Strike Rate analysis
- Filter by team

### 3. Bowling Leaderboard
- Top wicket taker: YS Chahal (213 wickets)
- Highest dot balls: YS Chahal (1,194)
- Economy rate analysis by bowler
- Total wickets across all bowlers: 12,923
- Filter by team

### 4. Team Analysis
- Mumbai Indians — highest runs & wickets
- Total Runs, Wickets & Sixes by team
- Inning-wise filter (1st & 2nd innings)

### 5. Match Insights
- Total Sixes: 13K
- Total Fours: 30K
- Sum of Extra Runs: 18K
- Dismissal types: Caught is most common
- Wides contribute 56.92% of all extra runs
- Mumbai Indians concede most extra runs

## 🛠️ Tools Used
- Power BI (Dashboard & Visualization)
- CSV Dataset (IPL ball-by-ball data)

## 💡 Key Insights
- V Kohli is the highest run scorer in IPL
- CH Gayle leads in total sixes
- YS Chahal is the top wicket taker with 213 wickets
- Mumbai Indians dominate in both runs and wickets
- Caught is the most common dismissal type
- Wides are the biggest source of extra runs (56.92%)
